## ‚ú® Features
* **Zero-config scan** ‚Äì walks the project tree and respects `.gitignore`, global ignores and custom `.aiignore`.
* **Smart snippets** ‚Äì keeps the first/last *N* lines of each text file; optional hex preview for binary/invalid-UTF-8 files.
* **Beautiful output** ‚Äì generated `CONTEXT.md` includes a directory tree, collapsible code blocks and language highlighting.
* **CLI ergonomics** ‚Äì powered by `clap` & `indicatif`; built-in `--help` and fancy spinner animation.
* **Pluggable roadmap** ‚Äì stubs for `diff` and `stats` sub-commands pave the way for incremental context and code analytics.

## ‚ö° Quick Start

```shell
# generate CONTEXT.md in project root (keeps 200 lines per file)
za gen

# advanced: keep 800 lines, include binary preview, write to docs/CONTEXT.md
za gen --max-lines 800 --include-binary --output docs/CONTEXT.md
```
